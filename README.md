# PR Reviewer Service (Go) — Тестовое задание

## Что реализовано
- CRUD-подобные операции: создание пользователей, команд, добавление участников.
- Создание PR: при создании автоматически назначаются до 2 активных ревьюверов из команды автора.
- Переназначение ревьювера (замена одного ревьювера на случайного активного участника его команды).
- Получение PR, merge (идемпотентно), список PR, назначенных пользователю.
- Статистика назначений (количество назначений по пользователям).
- Массовая деактивация команды с попыткой переназначения открытых PR.
